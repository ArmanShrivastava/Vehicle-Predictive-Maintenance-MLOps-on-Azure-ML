A production-grade machine learning platform that predicts vehicle component failure within a 30-day window using fleet telemetry data (engine temperature, oil pressure, battery voltage, brake wear, vibration, mileage). Built entirely on Azure Machine Learning with full automation from data ingestion to monitored production serving.
The platform implements a four-stage training pipeline (data prep → distributed LightGBM training → evaluation with AUC quality gates → conditional model registration) composed of versioned, reusable Azure ML components with end-to-end lineage tracking. Models are tracked via Azure ML's native MLflow integration, tuned with Bayesian hyperparameter sweeps with Bandit early termination, and served through Managed Online Endpoints with blue/green traffic-split rollouts for zero-downtime releases. Batch endpoints handle nightly fleet-wide scoring.
Production reliability is enforced through automated data drift monitoring (Jensen-Shannon distance on top features) with alerting, weekly scheduled retraining, and an Azure DevOps CI/CD pipeline with validation, training, and approval-gated deployment stages. Infrastructure follows enterprise patterns: scale-to-zero compute clusters, managed identities for credential-free storage access, private networking, and multi-workspace promotion via Azure ML Registries.
Stack: Azure ML (SDK v2/CLI v2), LightGBM, MLflow, Azure DevOps, Managed Online/Batch Endpoints, Azure Blob/ADLS, ACR, Key Vault, Application Insights
